# Mesaj Paneli — Hızlı Başlangıç

Ana rehber: [QUICKSTART.md](../QUICKSTART.md)

```bash
git clone https://github.com/bunyamindemir1/telegram-whatsapp-panel.git
cd telegram-whatsapp-panel
make setup
```

**http://localhost:8000** adresini açın. Dil: kenar çubuğundan veya `?lang=tr` ile Türkçe.

## 15 dil desteği

Arayüz İngilizce, Türkçe, Arapça (RTL), Rusça ve 11 dil daha destekler. Kenar çubudu → **Dil** menüsü.

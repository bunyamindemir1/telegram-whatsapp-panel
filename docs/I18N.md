## How users switch language

1. Sidebar → **Language** dropdown (persisted in `localStorage` + cookie)
2. URL query: `?lang=tr`
3. Browser `Accept-Language` on first visit

## For developers

### File layout

```
locales/
  en.json    # master keys (add new strings here first)
  tr.json    # full Turkish
  ar.json    # full Arabic (+ RTL)
  ...
static/js/i18n.js   # t('key'), applyI18n(), setLocale()
app/i18n.py         # server-side locale resolution + API
```

### Add a new string

1. Add key to `locales/en.json`
2. Add Turkish to `locales/tr.json`
3. Run `python scripts/validate_locales.py` (fills other langs from EN)
4. Use in HTML: `data-i18n="my.key"` or `data-i18n-placeholder="my.key"`
5. Use in JS: `tt('my.key')` or `toastT('toast.saved')`

### API

```http
GET /api/i18n/locales
GET /api/i18n/tr
```

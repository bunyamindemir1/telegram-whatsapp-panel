## Bug reports & features

Open a [GitHub Issue](https://github.com/bunyamindemir1/telegram-whatsapp-panel/issues/new/choose) with:

- OS and Docker/Python/Node versions
- Steps to reproduce
- Expected vs actual behavior
- Relevant logs (redact secrets and phone numbers)

## Security issues

Do **not** open public issues for vulnerabilities. See [SECURITY.md](../SECURITY.md).
